Project Plan Site

Site Name

Name: RB GamePit

RB is for my name, GamePit is because I want to give the page the feeling of a secret place to go and join the gaming world

Site Purpose

The site will probide a hub for getting information about videogames, it will be filtered by genre, date and rating, so it's easier to the user to choose a game to buy or to meet a new game.

Scenarios

I want to buy a game, but I can't choose one
You can navigate throug the game directory and find a game by genre, by rating and by release date. In the future it will alos be possible to look for best prices between all the stores.

I want to know about new games.
In the directory you will be able to find games that you didn't know, an also read some reviews from the comunity, In the comunity page you will be able to post and read about recommendations an tips about games.
